Description / Why did you choose this project?
This is a tool for people with dissociative disorders, especially DID and OSDD, to communicate with and keep track of other parts of their head that may not always be possible or easy in the way it is for many other folks! I chose this project because I have close friends with OSDD and other disorders which affect their ability to remember or keep track of thoughts.

What technology was used?
Most of this site's functionality is made of conversations between JQuery-Ajax and Flask. The server runs on Python code with Flask and an SQLAlchemy database. The client uses the standard HTML/CSS/JS with JQuery flavoring and many Ajax calls which are sent back to Flask routes.

What have you learned in this class?
I learned about a lot of great web programming methods I've never used before, a few of which I've used in this project - particularly JQuery and Ajax. These provide a really nice way to add code to webpages without the painful mess of raw Javascript, and allowing page updates and server calls without the need to wait for a refresh.
